# Legacy Desktop Application

This folder contains the desktop Fire Investigation Platform implementation.

Purpose:

- Preserve accepted functionality.
- Provide a functional reference for rebuilding the application.
- Identify existing workflows and field coverage.

These files are NOT the authoritative design.

When conflicts exist, the following documents take precedence:

1. FIP-Governance.md
2. PROJECT_INSTRUCTIONS.md
3. SHARED_SERVICES_ARCHITECTURE.md
4. Module Specifications.md

The objective is to modernize the desktop application into a Cloudflare-hosted Progressive Web Application while preserving accepted functionality where appropriate.
